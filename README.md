# Equal Sized Antonyms
A helpful list of antonyms with the same word length for your everyday variable names:

- `lo`/`hi`
- `up`/`dn`
- `rd`/`wr`
- `fd`/`bk`
- `rt`/`lt`
- `beg`/`end`
- `lhs`/`rhs` *(left/right hand side)*
- `src`/`dst`
- `top`/`mid`/`bot`
- `get`/`set`
- `old`/`new`⁰
- `min`/`max`²
- `new`/`del`¹⁰
- `add`/`del`¹
- `fix`/`var`
- `one`/`any`/`all`/`non`/`nil`
- `yay`/`nay`
- `head`/`tail`
- `less`/`more`
- `play`/`stop`
- `pred`/`succ`
- `prev`/`curr`/`next`²
- `push`/`pull`
- `this`/`that`⁰
- `yeah`/`nope`
- `east`/`west`
- `north`/`south`
- `start`/`pause`/`break`⁰
- `large`/`small`
- `accept`/`reject`
- `create`/`delete`⁰
- `insert`/`remove`
- `source`/`target`
- `unique`/`shared`
- `relative`/`absolute`
- `producer`/`consumer`

⁰ C++ keyword, cannot be used as an identifier  
¹ Python keyword, cannot be used as an identifier  
² Python builtin, should not be used as an unqualified identifier, but ok as member


## Equisized Non-Antonyms

- `get`/`got`/`gen` (`gen` for generate)
- `width`/`depth`
- `length`/`height`

## Oddballs
- `localhost`/`127.0.0.1` 😂
- `this->`/`other.`
